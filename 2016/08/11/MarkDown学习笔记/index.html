2.列表

无序列表

书写时：

* Red
* Blue
* Gary

生成后：

  • Red
  • Blue
  • Gary

有序列表

书写时：

1. Red
2. Blue
3. Orange

生成后：

  1. Red
  2. Blue
  3. Orange

4.链接

书写时：

This is a [Link Example](#) Link.

生成后：

5.强调

书写时：

*QD*
**QD**
_QD_
__QD__

QD
QD
QD
QD
